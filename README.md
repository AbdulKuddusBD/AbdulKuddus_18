# Abdul Kuddus Personal Portfolio

এই প্রোজেক্টটি একটা ব্যক্তিগত পোর্টফোলিও ওয়েবসাইট। এতে আপনার পরিচিতি, কাজের তথ্য, সার্ভিস, পোর্টফোলিও, এবং যোগাযোগের তথ্য দেওয়া আছে।

## ফাইলসমূহ

- `index.html` - প্রধান ওয়েবপেজ
- `style.css` - ওয়েবসাইটের স্টাইল ও ডিজাইন
- `README.md` - প্রোজেক্টের তথ্য
- ছবিগুলো - প্রোফাইল ও পোর্টফোলিও ইমেজ

## কিভাবে চালাবেন

1. `index.html` ফাইলটি ব্রাউজারে ওপেন করুন
2. অথবা লোকাল সার্ভার চালাতে পারেন, যেমন:
   - `Live Server` Extension (VS Code)
   - অথবা `python -m http.server` ব্যবহার করে

## ব্যবহার করা টেকনোলজি

- HTML
- CSS
